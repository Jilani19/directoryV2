# Assets Architecture

This folder contains the scalable frontend assets structure for the cGxP.Directory MVP application.

## Directory Structure & Usage Guidelines

### Home Page Images
- **Hero images:** `assets/images/home/hero/`
- **Trusted company logos:** `assets/images/home/trusted-companies/`
- **Statistics graphics:** `assets/images/home/statistics/`
- **Category icons:** `assets/images/home/browse-categories/`
- **Featured company logos:** `assets/images/home/featured-companies/`
- **Platform feature illustrations:** `assets/images/home/platform-features/`
- **Industry insight images:** `assets/images/home/industry-insights/`
- **CTA backgrounds:** `assets/images/home/cta/`
- **Footer assets:** `assets/images/home/footer/`

### Global & Common Images
- **Company logos:** `assets/images/company/logo/`
- **Country flags:** `assets/images/countries/flags/`
- **UI icons:** `assets/images/common/icons/`
- **Backgrounds:** `assets/images/common/backgrounds/`
- **Patterns:** `assets/images/common/patterns/`
- **Illustrations:** `assets/images/common/illustrations/`

### Other Sections
- **Directory Assets:** `assets/images/company/directory/`
- **Categories:** `assets/images/categories/`
- **Products:** `assets/images/products/`
- **Drugs:** `assets/images/drugs/`
- **Advertisements:** `assets/images/advertisements/`
- **Contact:** `assets/images/contact/`
- **About:** `assets/images/about/`
- **Auth (Login/Register):** `assets/images/auth/`

### Placeholders
- **User/Company Avatars:** `assets/images/placeholders/avatars/`
- **Banners:** `assets/images/placeholders/banners/`

### UI & Animations
- **Lottie Animations:** `assets/ui/animations/lottie/`
- **Videos:** `assets/ui/videos/`

### Base
- **Fonts:** `assets/fonts/`
- **SVGs (Standalone vectors):** `assets/svgs/`

## Instructions for Developers
- Do **not** upload raw, uncompressed images. Optimize all assets before placing them into these folders.
- Ensure all static UI images used by components map to their respective feature folders to maintain encapsulation.
- If a specific asset is used across multiple pages (e.g., logo, standard background), place it in the `images/common/` folder.
